# SC-TH 本科论文导出合规审查报告 v1

## 1. 审查目标

本轮审查目标是把 `scnu-thesis-portal` 从“结构化 Word 导出器”提升为“更接近华南师范大学本科论文正文审查稿格式基线的导出器”，并给出可复现的自动化与样例审查记录。

## 2. 规范依据

最高依据仅包括：

1. [《华南师范大学本科生毕业论文（设计）手册》](../sources/华南师范大学本科生毕业论文（设计）手册.doc)
2. [《华南师范大学本科毕业论文（设计）撰写基本规范》](../sources/华南师范大学本科毕业论文（设计）撰写基本规范.pdf)

若两者冲突，本轮默认采用 PDF 规范口径：目录为必备项，顺序采用“中文摘要 -> 外文摘要 -> 目录 -> 正文与注释 -> 参考文献 -> 附录 -> 致谢”。

## 3. 审查范围

- 输入 -> 解析 -> 预检 -> 结构建模 -> Word 导出 -> 合规检查 -> 渲染验证
- 仅审查 `.docx` 正文审查稿导出，不把学校正式统一封面纳入自动生成范围

## 7. 导出与自动合规检查结果

三份样例均成功导出，并通过 `scripts/check_docx_compliance.py` 自动检查。

### 自动检查摘要

- `sample-text-basic`：`PASS=13 / MANUAL_REVIEW=0 / NOT_SUPPORTED=2`
- `sample-docx-basic`：`PASS=13 / MANUAL_REVIEW=0 / NOT_SUPPORTED=2`
- `sample-docx-complex`：`PASS=13 / MANUAL_REVIEW=0 / NOT_SUPPORTED=2`

### 关键 PASS 项

- A4 页面尺寸
- 页边距与装订线
- 关键样式存在性
- 正文样式
- 中文摘要样式
- 外文摘要样式
- 目录字段存在
- 页眉内容与字体
- 页脚页码字段与字体
- 文档顺序
- 未发现伪造学校正式封面

### 当前 `NOT_SUPPORTED` 项

- 注释编号、页末注 / 篇末注的校规级自动保证
- 图题、表题位置与编号的全自动重建

## 8. 渲染验证

三份样例已通过 LibreOffice headless 成功转换为 PDF：

- `sample-text-basic.pdf`：7 页，A4
- `sample-docx-basic.pdf`：8 页，A4
- `sample-docx-complex.pdf`：8 页，A4

该步骤用于确认导出稿可稳定渲染，并辅助检查目录、页眉页脚、分页与页面尺寸。需要说明的是：**本轮人工抽检基于导出 OOXML、自动合规脚本和 LibreOffice 渲染结果；未在 Microsoft Word GUI 内逐页人工目测。** 这构成当前残余风险之一。

## 9. 人工复核结论

基于导出 OOXML、自动合规脚本和渲染结果，本轮可以确认：

- 页面设置、样式体系、目录字段、页眉页脚、页码和顺序均已进入可交付基线
- 不再伪造学校正式封面
- 外文摘要正文已切换为 `Times New Roman`
- 复杂 `.docx` 样例会在预检中显式暴露表格风险，不再静默吞掉

仍需人工复核的风险包括：

- 注释页末注 / 篇末注的严格规范化
- 图题 / 表题位置与编号
- 复杂富文本对象迁移
- 参考文献条目细项是否完全符合 GB3469-83
- 若题目包含副题，页眉是否需要进一步人工删减

## 10. 最终结论

**主体已达标，但存在若干需人工复核项。**

更具体地说：当前版本已经达到“可作为华南师范大学本科论文初稿 / 送审稿正文审查稿格式基线”的程度；但对于注释、图表题注、复杂 `.docx` 富文本迁移和参考文献细项，仍不能宣称“全自动严格合规”。这些限制已同步写入预检提示、README、质量清单与限制说明。
