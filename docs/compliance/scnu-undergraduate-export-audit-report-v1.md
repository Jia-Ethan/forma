# SCNU 本科论文导出合规审计报告 v1

## 审计范围

- 统一中间结构是否覆盖封面、摘要、正文、参考文献、附录、致谢、注释与复杂元素标记
- `.docx` 与文本两条入口是否共用同一口径
- 导出结果是否落地正式封面、目录字段、页眉页脚、页码与分节
- 文档口径是否已切换到新主线

## 审计结论

当前仓库已完成从“自由拼段落导出”到“规范驱动章节映射与模板渲染”的主线升级。

已确认：

- 正式封面已纳入主线第一页
- 页面顺序已固定为：封面 → 中文摘要 → 英文摘要 → 目录 → 正文 → 参考文献 → 附录 → 致谢
- 缺失内容不再自动补写
- TOC 已改为 Word 字段
- 页眉页脚与页码已改为按节控制
- 合规脚本不再把“疑似正式封面”当异常

## 仍保留的审计提醒

- 复杂 Word 元素默认进入人工复核
- 参考文献自动整理只覆盖有限格式细节
- Word 打开后仍建议手动触发一次目录更新并抽查分页
